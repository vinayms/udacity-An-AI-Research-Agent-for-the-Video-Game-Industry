# An AI Research Agent for the Video Game Industry

This project implements an AI-powered Research Agent designed to analyze trends, player sentiments, market dynamics, and technological advancements in the Video Game Industry. 

## Features
- **Market Trend Analysis**: Automated scanning of gaming news, reports, and release schedules.
- **Sentiment Mining**: Analysis of community discussions, reviews, and social media sentiment.
- **Competitor Intelligence**: Gathering and synthesizing data on competing game releases and features.
- **Report Generation**: Automatically compiling structured markdown reports for game studio stakeholders.

## Setup Instructions
1. Clone the repository:
   ```bash
   git clone git@github.com:vinayms/udacity-An-AI-Research-Agent-for-the-Video-Game-Industry.git
   ```
2. Install dependencies:
   ```bash
   pip install -r requirements.txt
   ```
3. Run the research agent:
   ```bash
   python main.py
   ```
